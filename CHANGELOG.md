# Changelog

All notable changes to HA-Powercard will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-11

### Added
- Initial release
- Distribution board style visualization
- Animated power flow lines using canvas
- Solar and grid power source display
- Main circuit breaker with power/energy/current data
- Multiple circuit breakers support (up to 8)
- Expandable device tables per circuit breaker
- Dark and light theme support
- Visual card editor for basic configuration
- Click-to-expand device tables
- Entity click for more-info dialogs
- HACS compatibility
- Responsive design for mobile devices

### Features
- Real-time power flow animation
- Particle-based flow visualization
- Active/inactive breaker status indicators
- Glowing busbar effect
- Configurable animation speed
- Unit of measurement auto-detection
